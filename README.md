# Clear Code

一键清理代码中的**注释**和**空行**，并支持代码格式化。

## 功能

### 删除注释和空行
- **Python**: 删除 `#` 单行注释、`"""..."""` / `'''...'''` 多行文档字符串、空行
- **C / C++**: 删除 `//` 单行注释、`/* ... */` 多行注释、空行
- 自动备份原始代码，支持一键恢复

### 代码格式化
- **C++**: K&R（Kernighan & Ritchie）风格缩进与花括号排版
- **Python**: PEP 8 官方规范格式化

### 恢复备份
- 一键还原到清理前的原始代码状态

## 使用方式

| 功能 | 快捷键 | 命令面板 |
|------|--------|----------|
| 删除注释和空行 | `Ctrl+Alt+/` | `Clear Code: 删除注释和空行` |
| 格式化代码 | `Ctrl+Alt+F` | `Clear Code: 格式化代码` |
| 恢复原始代码 | — | `Clear Code: 恢复原始代码` |

## 支持的语言

| 语言 | 文件扩展名 | 注释清理 | 代码格式化 | 格式化风格 |
|------|-----------|---------|-----------|-----------|
| Python | `.py` | `#` / `"""` / `'''` | ✅ | PEP 8 |
| C | `.c`, `.h` | `//` / `/* */` | ✅ | K&R |
| C++ | `.cpp`, `.hpp`, `.cc` | `//` / `/* */` | ✅ | K&R |
| Java | `.java` | `//` / `/* */` | ✅ | K&R |
| JavaScript | `.js`, `.jsx` | `//` / `/* */` | ✅ | K&R + 2空格 |
| HTML | `.html`, `.htm` | `<!-- -->` | ✅ | 标签缩进对齐 |
| CSS | `.css`, `.scss`, `.less` | `/* */` | ✅ | 选择器+属性缩进 |
